# 🚀 RSA Maps API Testing (Postman)

This repository contains a Postman collection for manual API testing of RSA Maps APIs.  
It demonstrates end-to-end testing of CRUD operations along with structured request handling.

---

## 📌 Endpoints Covered

- ➕ AddPlace (POST) – Create a new place  
- 🔍 GetPlace (GET) – Retrieve place details  
- ✏️ UpdatePlace (PUT) – Update existing place data  
- ❌ DeletePlace (DELETE) – Remove a place  

---

## 🛠️ Tools & Technologies

- Postman – API Testing  
- REST API – HTTP methods & request handling  

---

## ⚙️ Features

- Covers complete CRUD operations  
- Organized request structure for better readability  
- Demonstrates real-world API testing workflow  
- Easy to import and execute  

---

## ▶️ How to Use

1. Clone or download this repository  
2. Open Postman  
3. Click on Import and select the collection file  
4. Run the APIs individually or in sequence
